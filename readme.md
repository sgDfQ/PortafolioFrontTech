# PortafolioFrontTech

Portfolio front-end réalisé dans le cadre de ma montée en compétence en **intégration web (HTML/CSS)** avec **Front Technologies Experts**, à travers des exercices pratiques proches de cas réels.  
Ce projet met l’accent sur la **structuration HTML**, la **mise en page CSS**, l’**organisation des assets**, et l’ajout de **comportements JavaScript légers**.

> Encadrement / accompagnement : **Wassila AMOURA** (Front Technologies Experts).  
> Objectif : renforcer les bonnes pratiques front-end et produire un portfolio propre, lisible et maintenable. :contentReference[oaicite:1]{index=1}

---

## Aperçu

- **Type** : site statique (sans framework)
- **Technos** : HTML, CSS, JavaScript
- **Public cible** : démonstration d’intégration web / portfolio personnel
- **Démarrage** : ouverture directe du fichier `index.html`

---

## Fonctionnalités clés

- Structure HTML claire et sémantique (sections de portfolio)
- Styles CSS organisés (mise en page, typographie, responsive selon implémentation)
- Gestion d’assets (images, polices)
- Script JavaScript pour interactions/effets (ex. animation/écriture via `Writing.js`)

---

## Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**

---
